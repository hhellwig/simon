    Simon is a memory game that was created by Milton-Bradley in 1978. The physical version has colored panels that flash and make sounds to indicate the pattern they should be pressed in. When you make a mistake, you lose.

    We are only using the name Simon for education purposes (and fun). All credit for the original version of the game goes to Milton-Bradley. This program will not be used outside of this class.
